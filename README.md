# Arpit Bhojani — Professional Portfolio 

A modern, responsive, and performance-optimized portfolio website built to showcase expertise in Python development, Machine Learning, and Backend Engineering.

##  Key Features
* **Dual-Theme Architecture:** Seamlessly switch between a refined "Obsidian" dark mode and a "Paper" light mode.
* **Interactive UI:** Custom-built magnetic cursor tracking and smooth scroll-reveal animations using Intersection Observer API.
* **Performance First:** Zero heavy frameworks—built with vanilla CSS and JavaScript for near-instant load times.
* **Mobile-First Design:** Fully responsive grid layout that adapts from high-res monitors to mobile devices.
* **Live Work Integration:** Includes a dedicated "Daily Work" tracker and direct links to active research repositories.

##  Tech Stack
* **Frontend:** HTML5, CSS3 (Modern Flex/Grid), Vanilla JavaScript.
* **Typography:** Playfair Display (Serif) & DM Sans/Mono (Sans/Monospace).
* **Icons:** Simple Icons & FontAwesome SVG integration.
