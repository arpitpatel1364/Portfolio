About Me

I'm the kind of programmer who can't sleep when a system is broken — and can't stop building when an idea clicks.

It started with a simple question: what if software could understand the world the way we do? That pulled me from basic Python scripts into training real computer vision models, building secure backends, and finally — systems that think and reason. Not as a hobby. As a discipline.

My day-to-day is backend engineering in Django and Python — REST APIs, JWT auth, RBAC, database optimization. But I also build the AI layer on top: RAG pipelines, agentic systems with LangGraph, multimodal document processors, local LLM stacks. I want both sides to be production-grade, not just demos.

I commit to GitHub every single day — not to fill a streak, but because I genuinely work every day. I keep a detailed work log at AI-ML-at-Cactus-Creatives, build open-source tools, and document everything so others can learn from what I build.

Multimodal RAG — Most RAG systems only read. VisionScribe sees.

A production-grade multimodal RAG system treating every document page as both text and image. YOLO detects visual regions, Qwen-VL captions them, and everything is indexed into a unified hybrid search system combining dense and sparse retrieval.

Django RESTPyTorchYOLOv8Qwen-VLOllamaChromaDBPostgreSQLBGE-M3 · BM25 · RRFRAGASStreamlit
  • YOLO detects charts, tables & figures; Qwen-VL auto-captions every visual region
  • Hybrid dense + sparse retrieval (BGE-M3 + BM25) merged via Reciprocal Rank Fusion
  • Parallel ingestion via multiprocessing.Pool — 4× speedup on 50-page technical PDFs
  • Fully local stack — zero API cost, runs on-device with Ollama Llama 3
  • RAGAS evaluation: faithfulness, context recall & answer relevancy tracked
  • Django REST API with token auth, query logging, Streamlit UI & Docker support
  • Benchmarked at 16.1 pages/sec with 8 workers on M2 MacBook Pro
