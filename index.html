VisionScribe
Multimodal RAG — Most RAG systems only read. VisionScribe sees.

A production-grade multimodal RAG system that treats every document page as both text and image. YOLO detects visual regions, Qwen-VL captions them, and everything is indexed into a unified hybrid search system combining dense and sparse retrieval.

Django REST PyTorch YOLOv8 Qwen-VL Ollama ChromaDB PostgreSQL BGE-M3 · BM25 · RRF RAGAS Streamlit
  • YOLO detects charts, tables & figures; Qwen-VL auto-captions every visual region
  • Hybrid dense + sparse retrieval (BGE-M3 + BM25) merged via Reciprocal Rank Fusion
  • Parallel ingestion via multiprocessing.Pool — 4× speedup on 50-page technical PDFs
  • Fully local stack — zero API cost, runs on-device with Ollama Llama 3
  • RAGAS evaluation: faithfulness, context recall & answer relevancy metrics tracked
  • Django REST API with token auth, query logging, Streamlit UI & Docker support
  • Benchmarked at 16.1 pages/sec with 8 workers on M2 MacBook Pro
